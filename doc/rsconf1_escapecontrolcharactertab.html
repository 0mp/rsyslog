parser.escapeControlCharacterTab

Type: global configuration directive

Default: on

Description:

Replaces the tab character with the value of parser.controlCharacterEscapePrefix directive (defaults to #). parser.escapeControlCharactersOnReceive must be enabled for this to work.

Sample:

global ( parser.escapeControlCharacterTab="on" )

Legacy sample:

$EscapeControlCharacterTab on
