parser.spaceLFOnReceive

Type: global configuration directive

Default: off

Description:

Replaces the LF (line feed) character with a space when the message is received. This directive is at odds with parser.escapeControlCharactersOnReceive which would replace the LF character with the value of parser.controlCharacterEscapePrefix directive (defaults to #).

Sample:

global ( parser.spaceLFOnReceive="off" )

Legacy sample:

$SpaceLFOnReceive off
